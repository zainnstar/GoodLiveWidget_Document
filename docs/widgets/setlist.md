# セットリスト

歌枠配信向けの **曲順・現在曲・次曲** を OBS 上に表示するウィジェットです。

---

## 用途

- セットリストの表示
- 現在演奏中の曲名ハイライト
- 次の曲・過去の曲リスト
- 曲数進捗の可視化

---

## インスペクタ項目

### 曲リスト

| 項目 | 内容 |
| ---- | ---- |
| 項目一覧 | セットリストの曲名リスト |
| 曲を追加 | 新しい曲を末尾に追加 |
| 現在のインデックス | 現在曲の位置 |
| 全曲タイトルをコピー | クリップボードへエクスポート |

### 次の曲

| 項目 | 内容 |
| ---- | ---- |
| 次の歌を使う | 次曲リスト機能の ON/OFF |
| 次に歌う曲名 / 曲を追加 | 次曲キュー |
| 次の曲リストの前にテキストを付ける | プレフィックス |

### 見た目

| 項目 | 内容 |
| ---- | ---- |
| 現在の曲テキスト / 配置 | 現在曲の表示 |
| フォントサイズ | 現在曲 / セットリスト / 次曲 各種 |
| 縦位置オフセット | 現在曲・次曲・過去曲の Y 位置 |
| スクロール方式 | 往復スクロール / 先頭へジャンプ |
| シェイプ・色・枠線・影 | 装飾 |

---

## OBS 表示

- 現在の曲名を強調表示
- 必要に応じて次の曲・過去の曲をリスト表示
- 曲名の前に番号を付けるオプションあり

---

## 運用のヒント

- プロファイルごとにセットリストを保存すると、定期歌枠の準備が短縮できます
- インスペクタの **ウィジェット機能** から曲送り操作も可能（実装版による）
