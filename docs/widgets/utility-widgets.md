# その他ツール

進行補助・演出用の汎用ウィジェットです。  
いずれも **ウィジェット一覧** から追加し、**インスペクタ** でテキスト・色・フォント等を編集します。

---

## テロップ / 二重テロップ

| ウィジェット | 内容 |
| ------------ | ---- |
| テロップ | 1 系統のスクロールまたは固定テキスト |
| 二重テロップ | 2 系統のテロップを同時表示 |

用途: 告知、曲名、スポンサー名、ルール説明など。

---

## メモ

自由記述のテキストブロック。  
配信メモや TODO を OBS 上に常時表示する場合に使います。

---

## 時計

現在時刻を表示する壁時計ウィジェット。  
フォント・色・シェイプはインスペクタで調整。

---

## エンドロール

エンディング用のスクロールクレジット表示。  
曲リストやスタッフロールの自動スクロールに利用できます。

---

## ビンゴ / ルーレット / ダイス

| ウィジェット | 内容 |
| ------------ | ---- |
| ビンゴ | ビンゴカード・抽選進行 |
| ルーレット | ルーレット盤の抽選 |
| ダイス | サイコロの目を表示 |

視聴者参加型配信の進行補助向け。

---

## ストップウォッチ / タイマー

| ウィジェット | 内容 |
| ------------ | ---- |
| ストップウォッチ | 経過時間のカウントアップ |
| タイマー | カウントダウン |

---

## ペンライト

ネオン風ペンライト演出ウィジェット。  
ライブ会場風の視覚効果向け。

---

## パーティクル

パーティクルエフェクトを OBS 上に表示。  
演出の彩りとして利用。

---

## 共通操作

- **トランスフォーム** で OBS 上の位置・サイズを調整
- **プレビュー** で HTML 出力を事前確認
- プロファイル保存で配信ごとの preset 化
