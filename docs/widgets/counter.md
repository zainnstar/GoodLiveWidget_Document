# カウンター

数値（スコア、死亡回数、残り時間など）を OBS 上に表示するウィジェットです。

---

## 用途

- ゲーム配信のスコア表示
- 歌枠の曲数カウント
- 任意の数値 KPI 表示

---

## インスペクタ項目

### 操作

| 項目 | 内容 |
| ---- | ---- |
| 加算 / 減算 / リセット | 値を即座に変更（OBS に反映） |
| 値 | 現在の数値 |
| 増分 | 加算/減算ボタンの刻み幅 |

### 見た目

| 項目 | 内容 |
| ---- | ---- |
| ラベルテキスト | カウンター名 |
| シェイプ | 背景形状 |
| フォント | ラベル・数字用フォント |
| 文字色 / 背景色 / 枠線 / 影 | 配色・装飾 |
| 数字の配置 | 数字の位置（上下左右等） |
| フォントサイズ | ラベル・数字それぞれ |

---

## OBS 表示

- ラベルと数値を大きく表示
- WebSocket 経由で値変更が即時反映

---

## 複数配置

カウンターウィジェットは **複数追加可能** です。  
それぞれ独立した ID・値・見た目を持ちます。
