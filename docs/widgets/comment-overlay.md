# コメントオーバーレイ

ライブチャットを OBS 上に **装飾表示** するウィジェットです。  
配信連携（またはコメントテスト）から Hub 経由でイベントを受け取ります。

---

## 位置づけ

GoodLiveWidget 全体の **ウィジェットの 1 型** です。  
わんコメのような単体製品ではなく、カウンターやテロップと同じ基盤（WebSocket / HTML）で動作します。

配信者の主確認 UI は [配信連携パネル](../panels/streaming-integration.md) のライブログです。  
本ウィジェットは **視聴者向け OBS 演出** 向けです。

---

## インスペクタ項目

### 形状

| 項目 | 内容 |
| ---- | ---- |
| 形状 | コメントシェイプ（吹き出し形・配色） |
| 形状を適用 | 選択シェイプを反映 |
| コメントシェイプ一覧 | `/commentpreview` で形状比較 |

シェイプ JSON は `data/html/comment_shapes/` に格納されています。

### 表示要素

| 項目 | 内容 |
| ---- | ---- |
| コメント本文 / ユーザー名 / アイコン | 各 ON/OFF |
| スパチャ金額 | 有料コメントの金額表示 |
| 通常コメント / スパチャ・有料 | フィルタ |

### レイアウト

| 項目 | 内容 |
| ---- | ---- |
| 配置（縦／横） | コメントの並び方向 |
| 追加方向 | up / down / left / right |
| 最大表示件数 | 超過分はクリップ（スクロールバーなし） |
| フォントサイズ | 文字サイズ |

---

## データフロー

```text
YouTube / Twitch（またはコメントテスト）
    ↓
Streaming Event Hub
    ↓
CommentWidgetController
    ↓ WebSocket widgets[].comment
OBS ブラウザソース HTML
```

---

## プレビュー

- インスペクタ **このウィジェットをプレビュー** … 単体表示
- **コメントシェイプ一覧** … 形状カタログ（ズーム可）
- [コメントテスト](../panels/comment-test.md) … 実配信なしでイベント注入

---

## 複数インスタンス

コメントオーバーレイは **レイヤー内に複数追加可能** です。  
用途別（通常コメント用 / SC 専用など）に分けることもできます。

---

## 制限（初版）

- 流れる scroll アニメーションは静的スタック表示（意図的）
- チャット削除イベントによる DOM 除去は未対応
- Twitch 実配信は Phase 4 以降で拡張中
