# お絵描き

ImGui 上で描いた内容を OBS オーバーレイとして表示する **ペイントウィジェット** です。

---

## 用途

- 配信中のお絵描きコーナー
- 簡易ホワイトボード
- 視聴者への図解・文字入れ

---

## インスペクタ項目

### ツール

| ツール | 内容 |
| ------ | ---- |
| ペン | 自由描画 |
| 消しゴム | 描画の削除 |
| 塗りつぶし | 領域の塗りつぶし |

### パラメータ

| 項目 | 内容 |
| ---- | ---- |
| 色 | 描画色 |
| ブラシサイズ | 線の太さ |
| キャンバスサイズ | 描画領域の解像度 |
| キャンバスをリセット | 全消去 |

---

## OBS 表示

- キャンバス内容が WebSocket / HTML 経由で OBS に同期
- トランスフォームで OBS 上の表示位置・サイズを調整

---

## 複製時の注意

ペイントウィジェットを **Copy Selected** で複製すると、キャンバス内容もコピーされます。  
空のキャンバスから始めたい場合は新規追加してください。

---
