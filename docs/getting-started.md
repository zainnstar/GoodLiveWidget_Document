# クイックスタート

初回起動から OBS にオーバーレイを表示するまでの流れです。

---

## 1. アプリを起動する

`GoodLiveWidget.exe` を起動します。  
ImGui のドッキング UI が表示され、上部メニューと複数パネルが利用できます。

---

## 2. OBS キャンバス解像度を設定する

1. **[設定](panels/settings.md)** パネルを開く
2. **OBS キャンバス解像度**（幅・高さ）を、OBS のキャンバスと同じ値に設定  
   - 例: 1920×1080
3. **OBS WebSocket ポート** を確認（既定 **4455**）

この解像度は、接続パネルから OBS へドラッグしたときのブラウザソースサイズに使われます。

---

## 3. ウィジェットを追加する

1. **[ウィジェット一覧](panels/widget-list.md)** パネルを開く
2. レイヤー 1 を選択
3. 追加したいウィジェット（例: **カウンター**）を選ぶ
4. **[インスペクタ](panels/inspector.md)** で表示名・色・値などを編集

!!! tip "レイヤーについて"
    レイヤーは OBS のブラウザソース URL と 1 対 1 で対応します。  
    - レイヤー 1 → `http://127.0.0.1:21760/1`  
    - レイヤー 2 → `http://127.0.0.1:21760/2`  
    複数レイヤーに分けると、OBS 上で個別に位置・サイズを調整できます。

---

## 4. OBS にブラウザソースを追加する

### 方法 A: URL をコピー（手動）

1. **[接続](panels/connection.md)** パネルを開く
2. レイヤー横の **コピー** ボタンで URL をクリップボードにコピー
3. OBS で **ブラウザソース** を追加し、URL を貼り付け
4. 幅・高さを OBS キャンバス解像度に合わせる

### 方法 B: OBS へドラッグ（推奨）

1. 接続パネルの **OBS へドラッグ** ボタンを押したまま、OBS のソース一覧へドロップ
2. 設定パネルの解像度に基づいたサイズでソースが追加されます
3. OBS WebSocket が有効なら、ソースの自動更新も試みます

!!! warning "「シーンがアクティブになったときに再読み込み」"
    OBS のブラウザソース設定で **「シーンがアクティブになったときにブラウザを再読み込み」** にチェックを入れてください。  
    GoodLiveWidget を OBS より後から起動した場合、再接続時に表示が更新されます。

---

## 5. 表示を確認する

- OBS のプレビューでオーバーレイを確認
- アプリ内 **[プレビュー](panels/preview.md)** パネルで「ブラウザプレビューを有効」にすると、同じ HTML を ImGui 内で確認できます

インスペクタの **このウィジェットをプレビュー** で、選択中ウィジェットだけを拡大確認することもできます。

---

## 6. プロファイルを保存する

メニュー **ファイル → 保存** で、現在のウィジェット構成をプロファイル（JSON）として保存します。  
次回起動時は **前回のプロファイルを開く**（設定で選択可）で復元できます。
