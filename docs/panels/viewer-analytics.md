# 視聴者分析

`streaming.db` に蓄積した **視聴者集計データ** を閲覧・検索・CSV 出力するパネルです。  
コメント本文は保存されません（集計のみ）。

---

## 前提

- 配信連携ビルドが有効であること
- YouTube または Twitch に一度以上接続し、データが `streaming.db` に記録されていること

---

## 主な機能

| 機能 | 内容 |
| ---- | ---- |
| 配信チャンネル選択 | 監視したチャンネル（複数配信者）を切替 |
| 検索 | 表示名・チャンネル ID で絞り込み |
| 並び替え | 発言数 / 最終コメント / 配信参加回数 |
| 一覧表示 | 表示名、チャンネル ID、発言数、配信数、メンバー、最終コメント |
| CSV 出力 | 表示結果をエクスポート |

---

## 保存される情報（例）

- 視聴者のチャンネル ID・表示名
- コメント回数（本文なし）
- 配信への参加回数
- 来訪日・最終コメント日時
- メンバーシップ関連フラグ

---

## 配信者チャンネル分離

自分のチャンネルと、代行配信等の別チャンネルは **broadcast_channel_id** で論理分離されます。  
パネル上部のドロップダウンで切り替えてください。
