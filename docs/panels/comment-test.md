# コメントテスト

実配信に接続せず、**合成チャットイベント** を Streaming Event Hub へ送る開発・検証用パネルです。  
`streaming.db` には **保存しません**。

---

## 用途

- [コメントオーバーレイ](../widgets/comment-overlay.md) の表示確認
- スーパーチャット色・バッジ・初コメタグの動作確認
- 高評価数 / 登録者数ウィジェットへの metrics 送信

---

## プラットフォーム

| プラットフォーム | 状態 |
| ---------------- | ---- |
| YouTube | フル機能 |
| Twitch | 通常コメント等（一部イベントは YouTube 専用） |

---

## 主な設定項目

- 投稿者（表示名・チャンネル ID）
- 初コメフラグ（初@配信者 / 初@配信 / 初@今日）
- バッジ（モデレーター、メンバー、配信者、認証済み）
- イベント種別（通常 / SC 各種 / メンバーシップ / リアクション / メトリクス等）
- メッセージ本文

---

## ランダム送信

**ランダムコメント送信** で、通常コメントまたはスーパーチャットをランダムに生成します。  
コメントウィジェットの負荷・見た目確認に便利です。

---

## Hub ライブ ID

配信連携で接続中は Hub の `liveId` を使用します。  
未接続時はテスト用 ID（`TEST_LIVE_ID` 等）が使われます。
