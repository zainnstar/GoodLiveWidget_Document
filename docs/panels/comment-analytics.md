# コメント分析

`comment_archive.db` に保存された **コメント本文アーカイブ** を、配信（`live_id`）単位で閲覧するパネルです。

---

## 前提

1. 設定パネルで **コメントを DB に記録する** が **ON**
2. 配信連携でライブに接続し、コメントが発生していること

!!! info "既定は OFF"
    コメント本文の記録は **初期状態 OFF** です。  
    必要な場合のみ設定で有効化してください。

---

## 主な機能

| 機能 | 内容 |
| ---- | ---- |
| 配信チャンネル選択 | 監視チャンネルの切替 |
| 配信セッション選択 | `live_id` 単位でアーカイブを表示 |
| チャンネル ID 絞り込み | 特定視聴者の発言のみ |
| 本文検索 | キーワード検索 |
| 一覧 | 時刻、表示名、種別、本文、有料フラグ等 |

---

## 視聴者分析との違い

| | 視聴者分析 | コメント分析 |
| --- | ---------- | ------------ |
| DB | `streaming.db` | `comment_archive.db` |
| 内容 | 集計（回数・日付） | 本文・時刻・種別 |
| 既定 | 接続時に集計 | **記録 OFF** |
