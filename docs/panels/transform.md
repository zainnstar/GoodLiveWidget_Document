# トランスフォーム

レイヤー内ウィジェットの **位置・サイズ・回転** を編集するパネルです。  
OBS 上での見え方に直結するため、プレビューと併用すると調整しやすくなります。

---

## 役割

- 選択ウィジェットの X/Y 位置、幅・高さ、回転角の編集
- キャンバス上での視覚的な配置（ドラッグ操作）
- アスペクト比のリセット

---

## 主な操作

| 操作 | 内容 |
| ---- | ---- |
| 位置 | ピクセル単位で X/Y を指定 |
| サイズ | 幅・高さを指定 |
| 回転 | 度数で回転 |
| 比率リセット | サイズ変更後に縦横比を元に戻す |

インスペクタの **トランスフォーム** セクションと同じ値が表示されます。

---

## 背景色

設定パネルの **トランスフォーム背景色** で、このパネル上のキャンバス背景色を変更できます。  
編集時の視認性向上用で、OBS 出力には影響しません。

---

## レイヤー選択

ウィジェット一覧でレイヤーを切り替えると、トランスフォームパネルに表示されるウィジェットもそのレイヤーに限定されます。
