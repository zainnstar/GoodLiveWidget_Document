# インスペクタ

選択中のウィジェットの **内容・見た目・動作** を編集するパネルです。

---

## 役割

- 表示名の変更
- ウィジェット種別ごとのパラメータ編集（カウンター値、セットリスト曲名など）
- フォント・色・シェイプ（形状）の設定
- 単体プレビューの起動

---

## 共通項目

| 項目 | 内容 |
| ---- | ---- |
| 表示名 | ウィジェット一覧に表示される名前 |
| このウィジェットをプレビュー | CEF で選択ウィジェットのみ表示（`/monopreview`） |
| トランスフォーム | 位置・サイズ・回転（トランスフォームパネルと同期） |

---

## ウィジェット別の編集項目

インスペクタの内容はウィジェット種別によって変わります。

| ウィジェット | 主な編集項目 |
| ------------ | ------------ |
| [カウンター](../widgets/counter.md) | 値、増分、ラベル、色、フォント、シェイプ |
| [セットリスト](../widgets/setlist.md) | 曲リスト、現在曲、次曲、スクロール設定 |
| [コメントオーバーレイ](../widgets/comment-overlay.md) | シェイプ、表示要素、最大件数、フィルタ |
| [お絵描き](../widgets/paint.md) | ペン/消しゴム/塗りつぶし、色、キャンバスサイズ |
| その他 | 各ウィジェットの [ウィジェット一覧](../widgets/overview.md) を参照 |

---

## コメントシェイプ一覧

コメントオーバーレイ選択時、**コメントシェイプ一覧** ボタンから `/commentpreview` を開けます。  
吹き出し形状の比較専用画面で、プレビューパネルではズーム操作も可能です。

---

## 基本フォント

設定パネルの **基本フォント** で指定した PC 内フォントが、新規ウィジェット作成時の初期フォントとして使われます。
