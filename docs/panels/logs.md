# ログ

アプリ内の **操作ログ・通知** を表示するパネルです。

---

## 役割

- 接続状態の変化（OBS ドロップ、URL コピー等）
- 配信連携の接続/切断メッセージ
- エラーや警告の確認

---

## 操作

| 操作 | 内容 |
| ---- | ---- |
| ログをクリア | 表示中のログを消去 |

---

## 使い方のヒント

- OBS へドラッグが失敗した場合、WebSocket ポートや管理者権限の有無をログと [接続](connection.md) パネルで確認
- 配信連携ビルドが無効な場合も、ここに関連メッセージが出ることがあります
