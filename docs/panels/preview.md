# プレビュー

CEF（Chromium Embedded Framework）を使い、OBS と同じ HTML を **アプリ内でプレビュー** するパネルです。

---

## 有効化

**ブラウザプレビューを有効** にチェックを入れるとプレビューが表示されます。  
CEF がビルドに含まれていない版では「CEF が利用できません」と表示されます。

---

## プレビューモード

| モード | URL | 起動方法 |
| ------ | --- | -------- |
| 全レイヤー | `/` | **すべて表示** |
| レイヤー n | `/preview/{n}` | ウィジェット一覧でレイヤー n を選択 |
| 単体ウィジェット | `/monopreview` | インスペクタ「このウィジェットをプレビュー」 |
| コメントシェイプ一覧 | `/commentpreview` | インスペクタ「コメントシェイプ一覧」 |

!!! info "OBS との関係"
    プレビュー用 CEF と OBS ブラウザソースは **別インスタンス** です。  
    配信者向けの確認用であり、OBS 本番表示と完全同期ではありません。

---

## ズーム

**コメントシェイプ一覧モードのみ** ズーム（%）操作が可能です。  
通常のレイヤー/ウィジェットプレビューは **100% 固定** です。

---

## テーマ

設定パネルの Light/Dark テーマが、プレビュー背景およびコメントシェイプ一覧にも反映されます。

---

## 入力

コメントシェイプ一覧表示中は、マウスホイールで CEF 内スクロールが可能です（OSR イベント転送）。

---

## 枠線

設定パネルの **プレビュー枠線色・太さ** で、プレビュー領域の枠をカスタマイズできます。
