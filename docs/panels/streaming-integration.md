# 配信連携

YouTube / Twitch のライブチャットを取得し、アプリ内ライブログ・ウィジェット・DB へ配信するパネルです。

---

## 対応プラットフォーム

| プラットフォーム | 状態 |
| ---------------- | ---- |
| YouTube | 対応（StreamHost + YTLiveChat） |
| Twitch | 対応（IRC / TLS） |
| ニコニコ | 未実装 |

!!! note "同時接続"
    YouTube と Twitch を同時接続した場合、Hub の `liveId` は最後に接続した側で上書きされます。  
    実運用では **片方ずつ接続** を推奨します。

---

## YouTube 接続

### 入力モード

- チャンネルハンドル（`@...`）
- ライブ動画 ID
- チャンネル ID（`UC...`）

### オプション

- メンバーシップをログに表示
- メンバーシップを DB に記録

---

## Twitch 接続

| 項目 | 内容 |
| ---- | ---- |
| チャンネル | ログイン名（`#` なし） |
| IRC ニック | 空欄ならチャンネル名と同じ |
| OAuth トークン | `chat:read` 付き IRC 用トークン |

### オプション

- サブ / レイド通知をログに表示
- サブ通知を DB に記録

---

## ライブチャットログ

配信者が YouTube コメント欄の代わりに確認する **アプリ内ログ** です。

- 左右 2 分割（無料コメント / 有料系）
- 2 行表示（上: ユーザー情報・バッジ、下: 本文）
- スーパーチャットは金額ティアに応じた背景色
- 「○日ぶり」等の再訪タグ（DB 集計から）

---

## データの保存先

| データ | 保存先 | 内容 |
| ------ | ------ | ---- |
| リアルタイムイベント | Streaming Event Hub（メモリ） | ウィジェット・ライブログ向け |
| 視聴者集計 | `profiles/streaming.db` | 発言回数・来訪日等（**本文なし**） |
| コメント本文 | `profiles/comment_archive.db` | 設定 ON 時のみ |

---

## この配信（ランキング）

接続中の配信について、発言数ランキング等をパネル内に表示します（メモリ上、DB 非保存）。

---

## ビルド要件

配信連携が無効なビルドでは「配信連携はビルドされていません」と表示されます。  
配布版の StreamHost 同梱物を確認してください。
