# 設定

アプリ全体の環境設定を行うパネルです。変更内容は `userdata/config/environment.json` 等に保存されます。

---

## 言語

- 日本語 / English を切り替え
- 翻訳ファイル: `data/translations/ja.json`, `en.json`

---

## プロファイル（起動時）

| オプション | 内容 |
| ---------- | ---- |
| 前回のプロファイルを開く | 終了時のプロファイルを復元 |
| 新規プロファイルで開始 | 空の構成で起動 |

---

## 表示

| 項目 | 内容 |
| ---- | ---- |
| フォントサイズ | ImGui UI の文字サイズ |
| 基本フォント | 新規ウィジェットの初期フォント（PC 内フォントから選択） |
| テーマ | Light / Dark |
| トランスフォーム背景色 | トランスフォームパネルのキャンバス背景 |
| プレビュー枠線色・太さ | プレビューパネルの枠 |

---

## 接続

| 項目 | 既定値 | 内容 |
| ---- | ------ | ---- |
| WebSocket ポート | 21760 | HTTP / WebSocket サーバーのポート |
| OBS WebSocket ポート | 4455 | OBS ソース自動更新用 |
| OBS キャンバス解像度 | 1920×1080 等 | OBS へドラッグ時のソースサイズ |

---

## 配信連携 / コメント記録

| 項目 | 内容 |
| ---- | ---- |
| コメントを DB に記録する | **既定 OFF**。ON にすると `comment_archive.db` へ本文を保存 |

!!! warning "プライバシー"
    コメント記録を ON にすると、配信中のコメント本文がローカル DB に保存されます。  
    視聴者集計（`streaming.db`）とは別ファイルです。

---

## その他

- **ユーザーデータフォルダを開く** … `userdata/` をエクスプローラーで開く
- プリセット … OBS 解像度などのプリセット選択（実装による）
