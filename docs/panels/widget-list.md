# ウィジェット一覧

ウィジェットは OBS オーバーレイ上に表示する **個別の UI 要素** です。  
**ウィジェット一覧** パネルでレイヤーごとに追加・並べ替え・表示/非表示を管理します。

---

## レイヤーとウィジェット

- **レイヤー** … OBS ブラウザソース 1 枚に相当（URL: `/1`, `/2`, …）
- **ウィジェット** … レイヤー内に複数配置可能（カウンター、テロップなど）

同一レイヤー内のウィジェットは、トランスフォームパネルで位置・サイズ・回転を個別に設定します。

---

## ウィジェットの追加

1. ウィジェット一覧で対象レイヤーを選択
2. 追加ボタンまたはメニューから種類を選ぶ
3. インスペクタで表示名・パラメータを編集

---

## 主な操作

| 操作 | 内容 |
| ---- | ---- |
| 選択 | リストでクリック → インスペクタ・トランスフォームが連動 |
| 表示/非表示 | 目アイコンまたはチェックで切替 |
| 複製 | Copy Selected で同一設定のコピーを作成 |
| 削除 | 選択して削除（確認あり） |
| レイヤー移動 | 別レイヤーへドラッグまたは移動操作 |
| 並べ替え | レイヤー内の描画順（Z 順）を変更 |

---

## レイヤーの追加

ウィジェット一覧から **レイヤーを追加** できます。  
追加したレイヤー分、接続パネルに新しい URL が表示されます。OBS 側にも対応するブラウザソースを追加してください。

---

## プロファイルとの関係

ウィジェット構成は **プロファイル JSON**（`userdata/profiles/*.json`）に保存されます。  
歌枠用・ゲーム配信用など、用途別にプロファイルを分けて切り替えられます。
