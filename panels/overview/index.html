    パネル一覧

    GoodLiveWidget の UI は ImGui ドッキングパネル で構成されています。
    パネルの表示/非表示や配置は自由に変更でき、レイアウトは userdata/config/imgui.ini に自動保存されます。

    パネル一覧

    パネル 概要
    ウィジェット一覧 ウィジェットの追加・削除・レイヤー管理
    インスペクタ 選択中ウィジェットの詳細編集
    接続 HTTP URL・OBS ドラッグ、サーバー状態
    トランスフォーム レイヤー内ウィジェットの位置・サイズ・回転
    プレビュー CEF による OBS HTML プレビュー
    設定 言語・ポート・テーマ・プロファイル等
    配信連携 YouTube / Twitch 接続・ライブログ
    視聴者分析 視聴者集計 DB の閲覧・CSV 出力
    コメント分析 コメント本文アーカイブの閲覧
    コメントテスト 合成チャットイベントの送信（開発・検証用）
    ログ アプリ内ログの確認

    基本ワークフロー

    ウィジェット一覧で追加
    ↓
インスペクタで内容・見た目を編集
    ↓
トランスフォームで OBS 上の位置を調整
    ↓
接続パネルで OBS ブラウザソースを設定
    ↓
（任意）プレビューで確認

    共通操作

    パネルの表示切替

    メニュー 表示 から各パネルの ON/OFF を切り替えられます。
    ドッキングタブをドラッグしてレイアウトをカスタマイズできます。

    レイアウトの保存

    次のタイミングで ImGui レイアウトが 即時保存 されます。

    • パネル表示/非表示の切り替え
    • ウィジェットの追加・削除
    • ドッキング配置の変更

    言語切替

    設定パネルで 日本語 / English を切り替えられます。
    パネル内部 ID は英語固定のため、言語を変えてもレイアウトは維持されます。
